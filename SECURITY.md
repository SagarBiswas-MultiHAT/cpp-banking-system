# Security Policy

## Supported Versions

This project currently supports the latest `main` branch only.

## Reporting a Vulnerability

Please report security issues **privately**.

- Open a GitHub Security Advisory if available, or
- Email the maintainers listed in GOVERNANCE.md

Include:

- A clear description of the issue
- Steps to reproduce
- Affected versions or commit hashes
- Impact assessment if known

We aim to respond within **72 hours** and provide a fix or mitigation plan within **14 days**.
